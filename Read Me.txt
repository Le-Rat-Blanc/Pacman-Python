FR: Bonjour ! Pour lancer le jeu, il suffit de lancer l'executable "Pacman Python.exe" situé dans le dossier du même nom. Vous pouvez également consulter le code du jeu dans le dossier "Python Files". Attention : Pour lancer le jeu depuis le code d'origine, le module Pygame est necessaire. Vous pourrez le faire en lancant le fichier "Install Pygame.bat" si vous possedez Python3 sur votre pc et affilié a votre console/terminal.
